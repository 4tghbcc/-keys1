# Программа для стилистического преобразования чисел

## Описание
Программа для работы с датами рождения, которая выполняет следующие функции:
- Определение дня недели для заданной даты
- Проверка, является ли год високосным
- Расчет текущего возраста пользователя
- Стилистическое отображение даты в формате электронного табло

## Функциональность

### 1. Определение дня недели
Функция `get_day_of_week()` определяет, на какой день недели приходится заданная дата.

### 2. Проверка високосного года
Функция `is_leap_year()` проверяет, является ли год високосным.

### 3. Расчет возраста
Функция `calculate_age()` вычисляет текущий возраст пользователя на основе даты рождения.

### 4. Стилистическое отображение
Функция `create_digital_date()` отображает дату в формате электронного табло с использованием символов '*'.

## Использование

1. Запустите программу: `python date_transformer.py`
2. Введите последовательно день, месяц и год рождения
3. Программа выведет все результаты в удобном формате
